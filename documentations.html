Install CLI

Requirements

  • Mac OSX, Windows OS, Linux OS
  • Javascript ver 1.3 or higher
  • Node.Js ver 12 or higher

Install

Mac (Homebrew)

$ brew install momo-cli

NPM

$ npm install momo-cli

Yarn

$ yarn install momo-cli
